# Resume Project Description

## IntelliGen LMS — AI-Powered Learning Management System

**Role:** Full-Stack Developer (personal / portfolio project)  
**Stack:** Next.js 15, React 19, TypeScript, Tailwind CSS 4, Prisma, PostgreSQL, Vercel  
**Demo:** https://intelligen-lms.vercel.app

### Bullet points (copy-ready)

- Architected and built a production-style LMS with **87+ features** — courses, AI authoring, gamification, verifiable certificates, multi-tenant orgs, and role-based analytics — using Next.js App Router, Prisma, and PostgreSQL.
- Implemented an **AI layer** (course generator, quiz builder, learning assistant, coach) with OpenAI integration, instructor workflows, and graceful fallbacks when API keys are absent.
- Delivered **enterprise security and compliance** patterns: session auth, TOTP 2FA, OAuth SSO, device management, audit logs, and GDPR export/deletion flows.
- Built **engagement systems** — XP, badges, leaderboards, corporate training simulators, and quiz challenges — with demo-ready presentation modes for recruiter and client walkthroughs.
- Deployed to **Vercel** with documented CI/build pipelines, seed data, demo accounts, product tour, and showcase pages for professional portfolio presentation.
